# 🏓 中大運中桌球場空桌查詢系統

> 解決原預約系統需逐一查看場地的不便，提供一覽式儀表板與智慧篩選功能。

## 為什麼開發這個工具？ (Motivation)

在原本的預約系統中，若要尋找特定時段的空桌，我們必須**一個一個場地點進去確認**（如下圖），非常耗時且難以安排團體活動。

<img height="400" alt="image" src="https://github.com/user-attachments/assets/e918f2c4-3b66-4e5a-9e49-ca39507f072b" />

*（原本的介面：只能逐一查看，無法一眼掌握全場空桌狀態）*

## 核心功能 (Features)

本專案透過自動化爬蟲改善了這個體驗：

* ✅ **一覽式儀表板 (Dashboard View)**：
    * 將所有桌球場 (01-10) 的狀態整合在同一頁面，一目了然。
* ✅ **智慧篩選 (Smart Filtering)**：
    * 支援「時段」搜尋（例如：同時勾選 14:00 與 15:00）。
    * 系統會自動過濾出**該時段全程皆有空位**的桌號，省去人工比對的麻煩。
